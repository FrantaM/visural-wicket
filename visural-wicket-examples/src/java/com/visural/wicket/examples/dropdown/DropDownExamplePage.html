The Drop Down component works like a standard "Combobox" control, or as a "suggestion" control.

As a combobox, users can start typing and a matching list of records from the list will be displayed. Users may use the up and down arrow keys to select, or use the mouse to select an item form the list. In Combobox mode though, users must select a value.

In "Suggestion" mode, the users may free-text type a value, or select a value using the same Combobox-like behaviour.

The Combobox-mode uses the provided IModel (or backing CompoundPropertyModel) such that the value that is set is the list's Java object type itself.

In suggestion-mode the string value of the control is set against the provided model, since the user may free-text enter a value.
