InputHint is a behaviour which will allow you to insert a prompt into a blank text field in order to hint to the user as to it's purpose. When the user clicks in the field, the hint text will disappear making way for them to enter their own value.

When you create the input hint, you need to provide the form that the control is a part of, and a style for the hint (i.e. the content of the style='...' attribute).

You can optionally provide a "normal style" as well, if you are using element level style attributes in your markup.
