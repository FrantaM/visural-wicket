LessCSS extends CSS with variables, mixins, operations and nested rules.

You can find out more about LessCSS at http://lesscss.org/

LessCSS removes repetition from your stylesheets, and allows you to do complex cross-browser styling in one-line of code, using operations and nested rules.


You can include LessCSS support in your Wicket application using visural-wicket by adding this line to your Application.init()

getResourceSettings().setResourceStreamLocator(new LessCSSResourceStreamLocator(getResourceFinder()));

Then whenever you reference a file ending in ".less", it will automatically get parsed and translated by the LessCSS compiler.

add(new StyleSheetReference("stylesheet", BasePage.class, "style.less"));

This example application uses a "LessCSS" stylesheet, so download it and take a look to get started!
